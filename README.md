# MediaList

A personal CLI app for tracking different types of media

## Requirements

- A C++ compiler (e.g. g++ via MinGW, or MSVC)
- The save file is filled with test data and can be entirely deleted if need be. The program will create a new blank save file if one is not already found.

## Building

```
g++ main.cpp -o MediaList.exe
```

## Usage

```
MediaList.exe
```
